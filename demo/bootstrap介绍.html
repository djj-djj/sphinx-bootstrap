Bootstrap基础知识¶

Bootstrap是Twitter公司基于 HTML、CSS、JavaScript 开发的简洁、直观、强悍的前端开发框架，使得 Web 开发更加快捷。 Bootstrap提供了优雅的HTML和CSS规范，它即是由动态CSS语言Less写成，包括 Visual Studio Code 官网 等很多知名网站都使用其作为框架。
